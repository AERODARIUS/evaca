# STORY-20260311-004: Add dependency vulnerability scanning and patch cadence

## What was implemented
- Added workspace-wide vulnerability audit scripts:
  - `npm run security:audit:ci` (fails on high/critical)
  - `npm run security:audit:report` (JSON report output)
- Added CI gate in both main deployment workflows to execute `security:audit:ci` after dependency install.
- Added monthly scheduled workflow (`dependency-vulnerability-scan.yml`) that generates and uploads a dependency audit report artifact.
- Documented dependency patch protocol in `README` with:
  - severity-based SLA
  - ownership model (functions/web/shared)
  - emergency patch flow
  - rollback procedure
